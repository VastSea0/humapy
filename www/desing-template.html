Getting Started

Welcome to Hüma, a high-performance functional language designed for precision and kinetic speed. This guide will help you set up your environment and write your first industrial-grade Hüma program.

01 Installation

The Hüma toolchain is distributed as a single static binary. Use our shell script to automatically detect your architecture and install the huma-cli.

Terminal — Bash 
curl -fsSL https://huma.lang/install.sh | sh
info Architecture Support

Hüma currently supports x86_64 and ARM64 architectures natively. For legacy systems, please refer to the compilation-from-source documentation.

02 Quick Start

Verify your installation by checking the version. The Hüma compiler should respond instantly, reflecting our commitment to zero-latency developer tools. 

$ huma --version
Hüma Language Compiler v1.0.4-stable (build 4a92c)

03 First Program

Hüma syntax is designed to be expressive yet rigid. Below is a classic conditional loop demonstrating our core keywords: olsun (let), ise (if), and döndür (return).

example.huma 

// Define a simple function
işlem kontrolEt(değer) {
    olsun limit = 100;
    
    ise (değer > limit) {
        döndür "Limit aşıldı";
    } yoksa {
        döndür "Güvenli bölge";
    }
}

olsun sonuç = kontrolEt(150);
yazdır(sonuç);
warning Memory Safety

In Hüma, the olsun keyword creates an immutable binding by default. Use olsun mut if you require kinetic state mutation within a local scope.
